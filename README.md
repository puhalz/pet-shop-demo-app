# Pet Shop

A simple Spring Boot **Pet Shop API** with `GET` and `POST` endpoints for managing pets.

## 🚀 Tech Stack
- **Java 21**
- **Maven 3.6.3**
- **Spring Boot 3.5.0-M2**
- **Spring Web**
- **Spring Data JPA**
- **H2 Database (for testing)**
- **Mysql (for testing)**

## 📌 Features (/pet)
✅ Add a new pet - **POST**
✅ Get a list of all pets - **GET**
